Read the full job description: **[JOB.md](./JOB.md)**

## How to Apply

Open a [new issue](../../issues/new?template=application.yml) in this repo. Include:

1. **Your name and email**
2. **LinkedIn** (or equivalent)
3. **One skill, workflow, or example** of something you've built that shows you'd be great for this role

That's it. Use whatever agent or tool you want — Claude Code, Codex, OpenClaw, Cursor, or just do it yourself.

## Why This Way

Every is an AI-native company. Our team works with agents every day. If you can get an agent to apply for you, that tells us something. If you build something impressive to submit, even better.

This isn't a filter — it's a fast lane. Applications here go straight to the Head of Growth.
